# Proveedores

## Descripción

Catálogo de proveedores con quienes se realizan las compras de mercancía.

## Campos del formulario

| Campo | Descripción |
|-------|-------------|
| Nombre o razón social | Nombre del proveedor |
| RFC | Registro Federal de Contribuyentes; opcional |
| Correo electrónico | Email de contacto; opcional |
| Teléfono | Teléfono de contacto; opcional |
| Dirección | Dirección del proveedor; opcional |
| Estado | Activo o Inactivo |

## Uso en compras

Al registrar una compra, se selecciona el proveedor del catálogo de proveedores activos.

## Permisos requeridos

| Acción | Permiso |
|--------|---------|
| Ver proveedores | ver |
| Agregar proveedor | crear |
| Editar proveedor | editar |
| Eliminar proveedor | eliminar |
