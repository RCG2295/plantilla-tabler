# Mi Perfil

## Descripción

Permite a cualquier usuario autenticado actualizar su propia información personal, foto de perfil y contraseña, sin necesidad de permisos especiales.

## Información personal

| Campo | Descripción |
|-------|-------------|
| Nombre | Primer nombre del usuario |
| Apellidos | Apellidos del usuario |
| Correo electrónico | Email de acceso al sistema; debe ser único |
| Teléfono | Teléfono de contacto; opcional |
| Foto de perfil | JPG, PNG o WEBP, máximo 2 MB |

Al guardar la información, el nombre y foto se actualizan inmediatamente en el navbar sin necesidad de cerrar sesión.

## Cambio de contraseña

Para cambiar la contraseña se requiere:

1. Ingresar la contraseña actual (para verificar identidad).
2. Ingresar la nueva contraseña (mínimo 6 caracteres).
3. Confirmar la nueva contraseña.

El formulario de contraseña es independiente del formulario de información personal.

## Acceso

Todos los usuarios autenticados pueden acceder a su perfil. No requiere permisos especiales adicionales al de ver el módulo.
