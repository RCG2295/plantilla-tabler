# Unidades de Medida

## Descripción

Catálogo de unidades de medida utilizadas para los productos del inventario (pieza, kilogramo, litro, caja, etc.).

## Campos del formulario

| Campo | Descripción |
|-------|-------------|
| Nombre | Nombre completo de la unidad (ej. "Kilogramo") |
| Abreviatura | Símbolo corto (ej. "kg", "pza", "lt") |
| Estado | Activo o Inactivo |

## Uso en productos

La unidad de medida se muestra en el historial de movimientos y en los reportes de stock para indicar en qué magnitud se mide la cantidad.

## Permisos requeridos

| Acción | Permiso |
|--------|---------|
| Ver unidades | ver |
| Agregar unidad | crear |
| Editar unidad | editar |
| Eliminar unidad | eliminar |
